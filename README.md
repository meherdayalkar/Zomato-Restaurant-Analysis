# Zomato Restaurant Analysis - Bangalore

## Overview
Exploratory Data Analysis (EDA) of Zomato restaurant data from Bangalore using Python, Pandas and Matplotlib.

## Dataset
- 51,717 restaurants
- 17 columns (11 after cleaning)
- Source: Kaggle

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings
- BTM has the most restaurants in Bangalore (5,124)
- Lavelle Road has the highest average restaurant rating
- North Indian is the most popular cuisine
- 59% of restaurants accept online orders

## Steps Performed
1. Loaded and explored the dataset
2. Dropped irrelevant columns
3. Cleaned rate and approx_cost columns (converted from text to numeric)
4. Handled missing values
5. Performed exploratory analysis
6. Created visualizations
